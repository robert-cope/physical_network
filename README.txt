Files in this directory are relevant to the following manuscript:
Cope RC, Ross JV, Wittmann TA, Prowse TAA, Cassey P. (2016) Integrative Analysis of the Physical Transport Network into Australia. PLoS One. In Press.

Please cite this manuscript if/when you use these files, and contact Robert Cope (robert.cope@adelaide.edu.au) with any questions.

The physical_network directory contains an app using the R `shiny' package.
The csv files contain data relating to the physical network into Australia from other countries over time, which can be interrogated using the app.
The app also appears here: https://robertcope.shinyapps.io/physical_network
